# PWA Icon Placeholder

Untuk sementara, icon PWA belum ada. Ada 2 opsi:

## Opsi 1: Generate Online (Tercepat)

1. Buka https://www.pwabuilder.com/imageGenerator
2. Upload logo atau buat icon sederhana dengan background #008362 (hijau)
3. Download hasil generate
4. Copy `icon-192.png` dan `icon-512.png` ke folder `public/icons/`

## Opsi 2: Placeholder Sementara

Gunakan file text ini sebagai placeholder sampai icon asli siap.

Untuk testing, PWA bisa berfungsi tanpa icon (hanya warning di console).

## Icon Spec:

-   Size 1: 192x192 pixels
-   Size 2: 512x512 pixels
-   Background: #008362 (hijau Al-Anwar)
-   Icon: Cash register atau "KASIR" text (putih)
-   Format: PNG

Lihat file `ICON-GUIDE.md` untuk panduan lengkap membuat icon.
